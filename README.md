Разработайте объектно-ориентированную модель библиотечной системы. Библиотека содержит книги различных жанров (художественная литература, научная, учебная и т.д.), каждая книга характеризуется названием, автором, годом издания, ISBN и количеством экземпляров. Читатели могут брать книги на руки и возвращать их. Система должна поддерживать:

Регистрацию новых читателей и книг.
Выдачу и возврат книг с контролем сроков.
Поиск книг по различным критериям (автор, жанр, название).
Начисление штрафов за просроченный возврат с настраиваемыми правилами расчёта.
Система должна быть расширяема по типам изданий (журналы, газеты, электронные ресурсы) и правилам выдачи.

5 паттернов использовано:
Фасад - library
Билдер - PublicationBuilder
Фабричный метод - PublicationFactory
Шаблонный метод - Publication
Стратегия - SearchStrategy, FineCalculator, LendingRule
